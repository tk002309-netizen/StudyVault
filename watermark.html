ImgBB API Key (Thumbnail Upload)

ImgBB kya hai?
Subject thumbnail images ImgBB pe upload hoti hain — yeh free image hosting service hai. API key lene ke liye imgbb.com pe account banao → API section mein jao.
Yeh key subjects.html file mein IMGBB_KEY variable mein daalni hogi.
📝 Steps:
1. imgbb.com pe jao
2. Free account banao
3. Settings → API → API key copy karo
4. admin/subjects.html file mein line dhundo:
const IMGBB_KEY='your_imgbb_api_key_here'; Aur apni key wahan daal do.
